# Legacy Papers

This folder stores paper material used as reference for the application
design and methodological explanation.

## Files

- `Article_FASL_A39_final.docx`: source paper document kept as a legacy
  reference
- `Article_FASL_A39_final_extracted.txt`: raw text extraction created for
  quick local reading and search

## Notes

- The `.docx` file is the closest thing to the source document.
- The extracted text file is intentionally preserved as a raw helper
  artifact and may contain formatting noise from the extraction process.
- New explanatory material should be written as clean Markdown outside
  these legacy source files.
